こんな風にやってもうまくスペースが入らない場合。

```css
#hoge:after {
  content: " ";
}
```

<!--more-->
<h2>使える文字列</h2>
Unicode エスケープシーケンス(\uXXXX)形式を指定してやる必要があります。
スペースの場合は<code>\00a0</code>

```css
#hoge:after {
  content: "\00a0";
}
```
