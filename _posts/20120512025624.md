Java でのパーセント表示です。
メモ書きで何故 String なのかは忘れた。

<!--more-->

```java
int cntA = 63;
int sucA = 2;
String Prcnt = String.format("%.2f",( (double)sucA / (double)cntA * 100) );
// 0.0317460317
// ⇒ 3.17%
```
