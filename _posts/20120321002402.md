if の要素の判定ではなく has() を使う。

<b>悪い例：</b>
```
if($("親要素").children().hasClass("特定の要素")) {
    $("親要素").hover(function() {
         // ～～～
     }
}
```

<b>正解：</b>
```
$("親要素").has("特定の要素").hover(function() {
    // ～～～
}
```
