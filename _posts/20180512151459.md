## 概要

Nuxt.jsを利用したプロジェクトで、コンポーネント部分を[Element](https://element.eleme.io/#/en-US)にて担うことにした。

ただ、ビルド時に未使用のコンポーネントまでバンドルされてサイズも大きかったため、使用しているものだけバンドルさせるように調整を行った。

<!--more-->

## 方法

[babel-plugin-component](https://www.npmjs.com/package/babel-plugin-component)を利用する。

```
yarn add -D babel-plugin-component

// or

npm install -D babel-plugin-component
```

babel-plugin-componentをインストール後、Nuxtの`nuxt.config.js`に下記の設定を加える。


```language-js
build: {
    babel: {
      plugins: [
        [
          &#039;component&#039;,
          [
            {
              libraryName: &#039;element-ui&#039;,
            },
          ],
        ],
      ],
    },
}
```

`plugins/element-ui.js`を置き、下記のような記述にした。
