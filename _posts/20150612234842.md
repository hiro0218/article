## 原因

```
printf("\n");
```

`\n`と入力していたから。

### 対処

`￥マーク`じゃなくて、`＼（バックスラッシュ）`でないとダメ。

```
\n
```

## Macでバックスラッシュを入力する方法

<kbd>optionキー</kbd> + <kbd>￥キー</kbd>
