## 実装
### JavaScript

```language-javascript
document.addEventListener(&#039;contextmenu&#039;, function (e) {
    e.preventDefault();
}, false);
```

### jQuery

```language-javascript
$(document).on(&#039;contextmenu&#039;, function(){
  return false;
});
```
