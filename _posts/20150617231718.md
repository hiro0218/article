Java とかだとナチュラルに<code>split</code>で区切ってたんだけれど、C++にはそんなの無いみたい。つらい。
<!--more-->
<h2>方法</h2>
<h3>コード</h3>
<code>str</code>には"MADE IN JAPAN"みたいな感じでスペースが入った文字列が入ってる。
```language-c
vector<string> list;
stringstream ss(str);
string item;
while (getline(ss, item, ' ') && !item.empty()) {
    list.push_back(item);
}
```

これも`include`しておく。

```language-c
#include <vector>
#include <string>
#include <sstream>
```

---

これをcharで取り出すのは、こんな感じ。

```language-c
list[0].c_str();
```
