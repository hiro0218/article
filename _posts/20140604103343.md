## 実装

```language-css
.poping {
  transform: none;
  transition: transform 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}

.hide {
  opacity: 0;
  transform: translateY(-150%);
}
```

```language-js
let target = document.querySelectorAll(&quot;.poping&quot;);
let count = 1;

Array.from(target, el =&gt; {
  setTimeout(() =&gt; {
    el.classList.remove(&quot;hide&quot;);
  }, 400 * count++);
});
```
