## 実装

```language-css
.poping {
  transform: none;
  transition: transform 0.3s cubic-bezier(0.4, 0, 0.2, 1);
}

.hide {
  opacity: 0;
  transform: translateY(-150%);
}
```

```language-js
let target = document.querySelectorAll(".poping");
let count = 1;

Array.from(target, el => {
  setTimeout(() => {
    el.classList.remove("hide");
  }, 400 * count++);
});
```
