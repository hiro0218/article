<h2>状況</h2>

同一マシン内でGitHubアカウントが複数存在している状態。
（社用とプライベートでアカウントを分けている等）

<h2>コマンド</h2>

対象リポジトリへ移動して下記を実行。

<pre><code>git config --local user.name 名前
git config --local user.email メールアドレス
</code></pre>

確認

<pre><code>git config --local --list
...

user.name=名前
user.email=メールアドレス
</code></pre>
