<h2>状況</h2>
<p>同一マシン内でGitHubアカウントが複数存在している状態。<br />
（社用とプライベートでアカウントを分けている等）</p>
<h2>コマンド</h2>
<p>対象リポジトリへ移動して下記を実行。</p>
<pre><code>git config --local user.name 名前
git config --local user.email メールアドレス
</code></pre>
<p>確認</p>
<pre><code>git config --local --list
...

user.name=名前
user.email=メールアドレス
</code></pre>
