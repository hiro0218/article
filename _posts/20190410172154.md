## 状況

同一マシン内でGitHubアカウントが複数存在している状態。
（社用とプライベートでアカウントを分けている等）


## コマンド

対象リポジトリへ移動して下記を実行。

```
git config --local user.name 名前
git config --local user.email メールアドレス
```

確認

```
git config --local --list
...

user.name=名前
user.email=メールアドレス
```
