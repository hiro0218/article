jQuery とは違い、Javascript には trim 関数が無いので、
replace を使って空白をトリミング(取り除く)する。

```js
var str = str.replace(/(^s+)|(s+$)/g, "");
```

ちなみに jQuery はこんな感じ。

```js
var str = jQuery.trim(str);
```
