ぼーっと、こんな事やっちゃったので戒めとして…。

```java
Object obj = "オブジェクト";
String str = (String)obj;
```

<!--more-->

## Object を String に変換する

```java
Object obj = "オブジェクト";
String str = obj.toString();
```

## String を Object に変換する

```java
String str = "オブジェクト";
Object obj = str;
```
