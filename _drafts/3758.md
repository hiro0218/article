## 環境

- electron: 2.0.2


## 方法

```language-js
const controlKey = process.platform === 'win32' ? 'Ctrl' : 'Cmd';
const obj = {
  [`${controlKey}-B`]: {
    option: true,
    shift: true,
  },
};

// process.platform === 'win32' の場合
// obj の key 名が "Ctrl-B" になる
```
