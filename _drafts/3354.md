## コード

先述のサンプルは以下のような実装を行っている。

```language-html
&lt;div class=&quot;container&quot;&gt;
  &lt;ul class=&quot;item-list&quot;&gt;
    &lt;li class=&quot;list&quot;&gt;広ク会議ヲ興シ万機公論ニ決スベシ&lt;/li&gt;
  &lt;/ul&gt;
&lt;/div&gt;
```

```language-css
.container .list {
  margin: 0 calc(100% / 3);
}
```
