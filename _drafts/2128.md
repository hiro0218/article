<h2>概要</h2>
「1234」を「1200」にする。
<h2>方法</h2>
<h3>剰余（割り算の余り）</h3>
<pre class="language-"><code>var n = 1234;
n = n - n%100;</code></pre>

<h3>剰余（割り算の余り）</h3>
<pre class="language-"><code>var n = 1234;
n = Math.floor(n / 100) * 100;</code></pre>

<h3>正規表現</h3>
<pre class="language-"><code>var n = 1234;
n.replace(/(\d\d)\d\d/, "$1"+"00");</code></pre>
